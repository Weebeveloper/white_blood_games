# WhiteBloodGames

Github Pages site for `White Blood Games`, used to download build files.
[White Blood Games Website](https://weebeveloper.github.io/white_blood_games)

## Technicals

This project was generated with `Angular CLI version 16.2.16` and `node 16.14.0`, using `@angular/material 16`.

## Development

Run `npm run start`. The projects builds and deploys automatically on merge to `master`.

## Installation

Download the repository and run `npm i` or `npm ci`.
